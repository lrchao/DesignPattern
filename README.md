##设计模式

1. 单例模式
    - 懒汉式:该模式的特点是类加载时没有生成单例，只有当第一次调用 getInstance 方法时才去创建这个单例
    - 饿汉式:该模式的特点是类一旦加载就创建一个单例，保证在调用 getInstance 方法之前单例已经存在了
